# HabitFlow 中保真交互稿（V0.2）
版本：0.2（范围覆盖 V1 MVP）

## 0. 设计原则
- 少而稳：每屏最多 3 个一键选项；默认不滚动
- 解释优先：所有建议均配“为什么”与证据等级
- 低打扰：每日≤2 次推送，情境触发优先

## 1. 用户旅程（主线）
1) 登录/同意 → 入门评估（COM-B，约 12 题） → 选择首要目标（睡眠/步行/屏幕） → 生成“首胜挑战”（48h 内 2 次，每次≤5'）
2) 今日页：卡片栈（微目标 / 解释卡片 / 一键行动），底部导航：今日｜教练｜记录｜周报｜我
3) 教练对话：自然语言 + 插卡（解释卡片/微目标）
4) 周报：完成率、Top3 影响因子、下周微计划（1-2 条）
5) 隐私中心：授权矩阵 / 本地优先 / 一键清除

## 2. 屏幕规格（中保真布局）
### 2.1 今日页
- 顶部：日期 + “少而稳”标语；右上角“+目标”
- 卡片#1（今日微目标）
  - 标题：领域徽章（睡眠/步行/屏幕）+ 文案（≤22字）
  - 主按钮：现在就做（Primary）
  - 次按钮：换一个（Ghost）｜加入明天（Tertiary）
- 卡片#2（解释卡片）
  - 标题：因果一句话，尾随证据徽章（高/中/低）
  - 正文：60–100字，支持“展开更多原因”
  - 操作：采纳建议（Primary）｜了解原因（Link）
- 卡片#3（一键行动）
  - 3 个快捷：呼吸 2'｜台阶 2 组｜番茄 15'

### 2.2 教练（Chat）
- 输入区：占位“说说你今天的阻碍…”
- 消息气泡：系统（左深色）/用户（右浅色）
- 插卡：遵循卡片规范（有按钮、有说明）

### 2.3 周报
- KPI 条：完成率 + 较上周箭头
- Top3 因素：标签 + 影响描述（≤20字/条）
- 下周微计划：清单 + “一键采纳”按钮
- 底部：非医疗声明

### 2.4 入门评估
- 进度条：4/4 步
- 题型：单选/矩阵滑块（1–5）
- 输出：COM-B 雷达（简化）+ 初始微目标

### 2.5 隐私中心
- 授权矩阵：步数/睡眠/心率/位置/通知（逐项开关）
- 数据策略：本地优先、云端聚合（可一键清除）
- 导出：JSON/CSV

## 3. 组件规范（摘要）
- 证据徽章：High(#1C7D4E)/Mid(#946200)/Low(#8C8C8C)
- 卡片：圆角 16，内边距 16，卡间距 12，阴影 Elevation 2
- 字体：标题 18/24，正文字 15/22，按钮 16/20
- 可访问性：触控最小 44x44；对比度≥4.5:1；VoiceOver 标签完整

## 4. 空状态与错误
- 首日无数据：用“示例卡片”教学 + 引导授权
- 网络错误：对话与卡片缓存到本地，稍后重试
- 模型离线：提示“离线模式，仅提供通用建议”
