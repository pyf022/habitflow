## 3. AI 教练（Chat + 卡片流）
```
┌───────────────────────────────────────────┐
│ 你：最近午后很困，怎么提神又不影响睡眠？  │
│                                           │
│ 教练：                                     │
│ 数据→解释：过去 7 天你 15:00–17:00 步数偏低│
│ 且咖啡多在 17:30 之后。                    │
│ 微目标：15:00 台阶 2 组（约 2 分钟）。      │
│ 提醒：避免 17:00 后咖啡因。                 │
│ [ 一键加入今日 ] [ 替换为拉伸 2' ]          │
└───────────────────────────────────────────┘
[ 输入框…  ]  [ 发送 ]
```
**插卡规范：** 文本 + 行动按钮 + “为何这样判断？”弹层（展示简化依据）

## 4. 记录
```
- 自动：步数 / 活动分钟 / 睡眠（来自 HealthKit/Connect）
- 手动：情绪（1~5）/ 咖啡因 / 屏幕时长（若无系统权限）/ 备注
[ + 新增记录 ]
```

## 5. 周报（Weekly）
```
┌───────────────────────────────────────────┐
│ 本周复盘（第 14 周）                       │
│ 完成率： 62%  ↑                           │
│                                           │
│ Top3 影响因子：                            │
│ 1) 周末作息漂移 ↑ → 周一入睡延迟 ↑        │
│ 2) 晚屏幕↑ → 浅睡比例 ↑                    │
│ 3) 午后活动↓ → 下午主观疲劳 ↑              │
│                                           │
│ 下周微计划：                               │
│ - 22:20 洗漱 3'（睡眠）                    │
│ - 15:00 台阶 2 组（步行）                  │
│ [ 一键采纳 ]  [ 我想调整 ]                 │
└───────────────────────────────────────────┘
```
**说明：** 展示证据等级徽标；允许用户标记“有用/无感”做训练闭环


## 7. 情境触发（本地规则）
- 22:10–22:30 屏幕活跃 + 家中 Wi-Fi → 推送“睡前微目标”卡片（每日≤1 次）
- 14:30–16:30 连续久坐 ≥ 60 分钟 → 推送“2 分钟台阶/拉伸”
- 用户选择的“安静模式”时段内不推送
